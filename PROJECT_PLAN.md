# Project Plan

## Phase 1: Design Foundation

Define the AI organization, first agent roster, operating loop, memory model, and first implementation slice.

## Phase 2: Project Scaffold

Choose the app stack, scaffold the project, and create a first working screen.

## Phase 3: Operator Cockpit

Build the chat-first cockpit with agent roster, task queue, memory panel, product vault, and output workspace.

## Phase 4: Agent Router

Implement request classification, agent selection, task creation, memory updates, and human approval gates.

## Phase 5: Specialist Agents

Add V1 agent behaviors for Dean, Ledger, Forge, Voice, Scout, and Launch.

## Phase 6: Trading Evidence Workflows

Add structured flows for trade review, bot research, backtest summaries, readiness gates, and productized education assets.

## Phase 7: Persistence And Integrations

Add durable storage, file imports, export flows, and later integrations with MT5, content channels, or payment systems.
